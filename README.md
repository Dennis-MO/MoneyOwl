# Next.js Application

This is a Next.js project bootstrapped with essential configurations.

## Getting Started

### Prerequisites
- Node.js v20.15.1 or later
- npm v10.7.0 or later (installed automatically with Node.js)

### Installation

All dependencies have been installed. To verify:

```bash
npm list next react react-dom
```

### Running the Development Server

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Available Commands

- `npm run dev` - Start the development server
- `npm run build` - Build the application for production
- `npm start` - Start the production server
- `npm run lint` - Run ESLint to check code quality

## Technologies Included

- **Next.js 14**: React framework for production
- **React 18**: JavaScript library for building UIs
- **TypeScript**: JavaScript with static types
- **Tailwind CSS**: Utility-first CSS framework
- **ESLint**: JavaScript linter for code quality
- **PostCSS & Autoprefixer**: CSS preprocessing
